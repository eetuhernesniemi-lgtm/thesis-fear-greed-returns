HOW TO RUN - Thesis Analysis
==============================
1. Upload the following three files to Google Colab:
   - edgar_panel_data.xlsx  (financial statement data from SEC EDGAR)
   - fear_greed_index.csv   (CNN Fear & Greed Index daily data from Kaggle, Patel 2024)
   - sp500_monthly_returns.csv  (monthly stock returns, built by download_stocks2.py)

2. Run all cells in order (Runtime → Run all):
   - Chunk 1: Install libraries
   - Chunk 2: Load all data (EDGAR, Fear & Greed, stock returns, market, VIX)
   - Chunk 3: Build monthly stock return panel
   - Chunk 4: Merge EDGAR financial data (with look-ahead bias correction)
   - Chunk 5: Construct derived variables (leverage, book-to-market, momentum)
   - Chunk 5b: Descriptive statistics (Table 2)
   - Chunk 6: Run regressions (Year FE + double-clustered SEs, Table 3)

3. sp500_monthly_returns.csv was generated by running download_stocks2.py
   locally in Python before the analysis. This script downloads monthly
   adjusted closing prices for all S&P 500 companies from Yahoo Finance
   using the yfinance library and saves them as percentage returns.
